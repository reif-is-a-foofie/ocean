# Project Plan & Agent Handoff Log

This file is the single source of truth for agent progress, handoff, and task status. Every agent must declare intent before acting, and log completion after each atomic task.

## Task Structure
- **Status:** `pending` | `in_progress` | `completed`
- **Owner:** Agent name
- **Description:** What is being done and why
- **Acceptance Criteria:** What counts as "done"
- **Timestamp:** ISO8601
